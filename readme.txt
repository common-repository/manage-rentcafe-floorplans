A simple way to manage floor plans through the RentCafé Floorplans API.

== Description ==

Requires the installation of Advanced Custom Fields. Once our recommended fields are installed into your website, you can enter your RentCafé Marketing API key and then the plugin will automatically import the floorplans from your property! All floorplans can be found under "Floorplans" in the left menu. Place floorplans in your pages, posts, etc using shortcode "[floorplans]".

== Installation ==

1. Download the zip folder named  manage-rentcafé-floorplans.zip
2. Unzip the folder and put it in the plugins directory of your wordpress installation. (wp-content/plugins).
3. Activate the plugin through the plugin window in the admin panel.
4. Install the Advanced Custom Fields plugin, and import our recommended custom field group. 
5. Enter your RentCafé marketing API key and also the property code for you property. 
6. In your Wordpress Dashboard, go to "Floorplans" in the left Wordpress menu. You should immediately see floorplans in the CMS. 
7. Copy the shortcode and paste it into a page, post or widget.
